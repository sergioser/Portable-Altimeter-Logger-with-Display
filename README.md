Portable-Altimeter-Logger-with-Display
======================================

Displays maximum altitude using the BMP 180.

Hooking up the hardware:


for the BMP 180,

    3.3V to VCC (5v will fry the BMP 180)
    GND to GND
    Analog 4 to SDA
    Analog 5 to SCL



For the display,

in the format Microcontroller Pin to Bubble Display pin:

    D8 to P1
    D5 to P10
    D11 to P4
    A1 to P6
    D14 to P12
    D6 to P11
    D10 to P3
    D3 to P8
    D9 to P2
    D4 to P9
    A0 to P7
    D12 to P5
